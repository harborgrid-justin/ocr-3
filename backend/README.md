# OCR App Backend

This is the backend server for the OCR App.

## Getting Started

1. Install dependencies: `npm install`
2. Start the server: `npm start`

## API Routes

- `/api/ocr`: OCR-related routes
- `/api/nlp`: NLP-related routes
- `/api/storage`: Storage-related routes
